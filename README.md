# battle_ope_recorder
electronアプリ
# 行ったこと
    接続されているビデオデバイスの名前とIDをPython(flask)上で取得
    デバイス名をセレクトできるようにした
    デバイスIDをPOSTで送れるようにした
# 行うこと
    ビデオの再生 (60fps位でるか確認)
